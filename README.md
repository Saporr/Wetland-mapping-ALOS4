# Wetland Mapping using ALOS-2/ALOS-4 and Deep Learning

This repository contains the code used in the study:
"A deep learning approach in wetland mapping using ALOS-2 and ALOS-4 SAR data"

## Contents
- Data preprocessing
- CNN models

## Data
- Sentinel-2: publicly available from ESA
- ALOS-2/ALOS-4: provided by JAXA EO-RA4 (not publicly available)

## Requirements
pip install -r requirements.txt

## Reproducibility
Sample data and scripts are provided to reproduce key experiments.
